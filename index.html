Housing Mortgage Calculator

Mortgage Loan is used either by purchasers of real property to raise funds to buy real estate, or alternatively by existing property owners to raise funds for any purpose, while putting a lien on the property being mortgaged.

The user required to input the details of their mortgage loan, for example: property price, loan-to-value and mortgage years. which is going to determine a mortgage's monthly payments. It also provides a table for repayment timeline.

Start Now!

Mortgage Amendment Repayment

The user required to input the details of their mortgage loan, for example: original mortgage amount, interest rate and original mortgage period which is going to determine that when the user wants to do the partially prepayment or repay all owing value the mortgage payments.

Start Now!

Mortgage Stress-Test

The user required to input the details of their mortgage loan, for example: monthly household income, loan tenor,interest rate etc. which is going to determine that will the bank do a mortgage loan for home buyers. Based on the Debt-to-Income ratio (DIT), the percentage will be calculated to decide the application will pass or not. The system assumed the interest rate increased 3% for evaluating the stress-test.

Start Now!

Housing Adviser Guide

A tool used to determine that the amount of budget that the user can afford. Based on the personal financial situation and housing requirements to generate a customized plan for the homebuyers. The system will guide you step by step for the evaluation.

There are 10 stages for evaluating the user's financial situation and personal requirements.

It will take about 10-15 minutes for this evaluation.

Start Now!
